# サービス概要
未経験エンジニア・新人エンジニにおすすめの参考書を投稿し、紹介するサービです

# 使用技術
- バックエンド
  - PHP 8.1
  - Laravel 10.10
- フロントエンド
  - Tailwindcss
  - Breeze
- インフラ
  - Docker
  - Nginx
  - MySQL8
